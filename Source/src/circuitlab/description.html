A benchmark for Petit Poucet

Petit Poucet is Java library that allows users to create function circuits over which lineage queries can then be addressed.

This benchmark measures the impact on performance (processing time and memory) of the lineage features. It does so by evaluating various function circuits on multiple types of inputs.

Set of problems

The problems correspond to simple-to-medium computing tasks.

  • Window average
  • Sum of triangle areas
  • Nested bounding boxes
  • Get all numbers

Prerequisites

The self-contained JAR file is a bundle of the following dependent libraries (which you don't need to install):

  • Azrael
  • LabPal
  • Petit Poucet

How to run this lab

This lab can be run by typing, at the command line:

java -jar petitpoucet-circuit-lab.jar --autostart
